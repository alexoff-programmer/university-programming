# Архивация учебных проектов по дисциплине «Программирование» (C++)

В данном репозитории аккумулированы практические и лабораторные работы, выполненные в рамках университетского курса. Основная цель материалов — демонстрация процесса освоения базовых концепций Computer Science и навыков разработки на языке C++ в среде Visual Studio.

### Структура репозитория

Содержимое разделено по тематическим модулям, соответствующим этапам учебного плана:

* **Основы алгоритмизации (каталог `first/`)** Начальный этап обучения: работа с управляющими конструкциями, логическими операторами и базовым синтаксисом. Включает серию последовательных задач (от `first.cpp` до `eight.cpp`).
* **Алгоритмы сортировки и работа с массивами (`BubbleSort/`, `Сортировка тест/`)** Реализация классических алгоритмов упорядочивания данных. Основное внимание уделено методу «пузырька» и принципам оптимизации перебора элементов в массиве.
* **Типы данных и структуры (`Structs.cpp`, `4Lec.cpp`)** Переход к пользовательским типам данных. Работа со структурами (`struct`), организация связанных данных и изучение материалов лекционного курса.
* **Рекурсивные алгоритмы (`Рекурсия/`)** Проектирование функций с самовызовом. Решение задач на вычисление последовательностей и обход древовидных структур.
* **Подготовка к аттестации (`GettingReady.cpp`)** Сводный файл с решениями типовых задач, использованный для итогового повторения материала перед зачетом.

### Техническая информация

Проекты разработаны и скомпилированы в среде Microsoft Visual Studio. Для удобства проверки и запуска в репозитории сохранены конфигурационные файлы решений (`.sln`) и проектов (`.vcxproj`).

Служебные файлы компиляции (логи, объектные файлы и кэш-директории `x64/Debug`) включены в архив для обеспечения полной идентичности состояния среды на момент последней успешной сборки.

### Назначение

Репозиторий служит цифровым подтверждением компетенций в области процедурного программирования и может быть использован как база для повторения основ разработки на C++.
